React.lazy() y componente Suspense

-No es necesario cargar e importar cada componente en toda la página, sino que las importaremos solo cuando
    las necesitamos.
-Para eso usaremos el metodo lazy() de react.

Uso de metodo lazy:
    -Ya no importaremos el componente como siempre, sino que guardaremos en una const el metodo lazy().
    -Como parametro tiene una función, esta debe retornar un import dinamico con el componente a usar.
    -Para el import dinamico, solo se usa el metodo import() con la ruta.
    -Para que esto funcione, el componente que va a usar este componente con el import dinamico debe estar 
        encapsulado por un componente de react llamado 'Suspense' .
    -Suspense necesita un atributo fallback, este tendrá un componente o elemento html que renderizará mientras
        el componente principal se renderiza.
    -Finalmente, el componente que se va a traer mediante el import dinamico, debe exportarse de forma defaul,
        de lo contrario no servirá.

-En nuestro ejemplo, en el App, lo hacemos solo con el componente Favs, pero lo ideal es hacerlo con todos.
