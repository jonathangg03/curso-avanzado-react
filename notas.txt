Mutaciones para iniciar sesión

-La Api de GraphQL esta programada para que, sí hacemos el login, y tenemos el password que no va con el 
    email, nos devuelva un error de password incorrecto.
-La estructura del mutation es la misma que la de register.
-El uso y render prop son igual que en el de registro.

Nota: Para probar el login, y los errores y carga de la clase pasada, se registró el usuario:
    user: jona@gmail.com
    pass: 1234
