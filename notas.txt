¿Qué es CSS-in-JS?

-Antes teniamos separadas cada funcionalidad en HTML CSS y JS (HTML-centric).
-Luego, podiamos tener dentro de un JS un render, que retornara HTML (JSX) y llamará a un archivo css (JS-centric).
-Ahora, pasamos a CSS-in-JS, nos permite no solo hacer el jsx si no colocar los estilos en el mismo JavaScript, 
    de la siguiente manera:
        const Button = styled.button`
        border-radius: 4px;
        ${props => props.accent && `
            background: red;
            color: #fff;
        `}
        render () {
            return (
                <Button>
                    Click here!
                </Button>
            )
        }
-Con CSS-in-JS no debemos preocuparnos por tener dos clases igual, no nos debemos preocupar por el Critical Path 
    CSS, tendremos un CSS mucho más óptimo.
-Para esto, usaremos styled-components, biblioteca para CSS-in-JS pensada más que todo para React.
